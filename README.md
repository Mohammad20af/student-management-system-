# Student Manager

A simple command-line Student Management System in Python. Manage student records with names, ages, grades, and calculate averages.

## Features

- **Add Student** – Add a new student with name, age, and grades
- **Show Students** – Display all students in the system
- **Search Student** – Find a student by their ID
- **Calculate Average** – Calculate and display average grades for all students
- **Delete Student** – Remove a student by ID
- **Update Student** – Modify student information

## Requirements

- Python 3.6 or higher (uses only the standard library)

## Setup

1. Clone or download this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/student-manager.git
   cd student-manager
   ```

2. No extra packages needed; run with Python 3.

## Usage

Run the program:

```bash
python main.py
```

Then select an option from the menu:

1. Add Student
2. Show Students
3. Search Student by ID
4. Calculate average
5. Delete Student
6. Update Student Data
7. Exit

## Project Structure

- `Student.py` – Contains the `Student` class and all management functions
- `main.py` – Main program with menu loop
